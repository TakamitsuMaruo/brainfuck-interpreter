# Brainfuck-interpreter

## Overview  
A Brainfxxk interpreter.  

## USAGE   
`$ win-32 <your-script>`  
`$ win-64 <your-script>`  
or Compile `bf-i.c` and `$<your-bf-interpreter> <your-script>`   


# SAMPLE SOURCE  

## Overview  
Print A~Z and '\n'.  


## USAGE  
`$ win-32 sample.bf`  
`$ win-64 sample.bf`
